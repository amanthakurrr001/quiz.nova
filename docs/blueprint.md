# **App Name**: QuizGenius

## Core Features:

- Gmail Authentication: Secure user authentication via Gmail, including OTP verification and account setup.
- User Profile Creation: Capture user information (Name, Profession, Age) during onboarding to personalize the experience. This step will happen once after Gmail verification.
- Gemini API Key Management: Allow users to input and manage their Gemini API key for AI quiz generation. API keys will be tied to the currently logged-in user.
- AI Quiz Generation: Generate quizzes using the Gemini API based on user-provided topics, number of questions, and difficulty levels.
- Manual Quiz Creation: Provide a form-based interface for users to create quizzes manually, including questions, options, and correct answers.
- Quiz Output and Actions: Display the generated or manually created quiz with options to save, download, or play the quiz.
- Quiz History and Growth Tracking: Allow users to review their past quizzes and track their progress via a graphical view.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to evoke trust and intelligence.
- Background color: Light grey (#F0F8FF) to ensure a clean and focused user experience.
- Accent color: A lively green (#90EE90) to indicate success and progress.
- Body and headline font: 'Inter', a sans-serif font, for a modern, neutral, and readable interface.
- Use clear and intuitive icons for actions such as 'Save,' 'Download,' and 'Play Quiz'.
- Implement a clean, responsive layout to ensure usability across different devices.
- Employ subtle animations to provide feedback and enhance user engagement, such as loading animations during quiz generation.
